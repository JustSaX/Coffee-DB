Principles:
- Don't repeat yourself!
- Functions should only do one thing
- Use classess only when state is needed

How to structure my programm:
-  keep all ionic elements together (e.g. in home only ionic elements)
-  in cases ionic elements are created in functions or classess (e.g. by interation)
    put them in a separate file
